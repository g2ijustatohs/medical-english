🚀 GitHubへの反映手順（ターミナル不要版）

1

「書き出し」タブで words.js をダウンロードする

2

GitHubリポジトリを開く

https://github.com/g2ijustatohs/medical-english
3

words.js をクリック → 右上の鉛筆アイコン → 全選択（Cmd+A）→ 削除 → ダウンロードしたファイルの中身を貼り付け → Commit changes

4

約30秒後に https://g2ijustatohs.github.io/medical-english/ に反映されます

⚡ ターミナルで一発デプロイ（上級・推奨）

1

まだやっていなければ、GitHubリポジトリをMacにクローン：

git clone https://github.com/g2ijustatohs/medical-english.git ~/medical-english-repo
2

このツールでダウンロードしたwords.jsをリポジトリフォルダに上書きコピー：

cp ~/Downloads/words.js ~/medical-english-repo/words.js
3

一発デプロイ（以下をターミナルに貼り付け）：

cd ~/medical-english-repo && git add words.js && git commit -m "単語追加 $(date '+%Y-%m-%d')" && git push
4

初回のみ認証が必要です。GitHubのユーザー名とPersonal Access Tokenを入力してください（パスワードではなくToken）。

🔑 Personal Access Token の取得方法

1

GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token

2

「repo」にチェックを入れてトークンを生成。表示されたトークンをコピーして保管してください（再表示されません）。
